README.md – Morse Code
Introduction
Morse code is a communication system that uses dots (.) and dashes (-) to represent letters and numbers.
History
Morse code was created by Samuel Morse and Alfred Vail in the 1830s to send messages over long distances using telegraphs.
How It Works
Each letter has a unique pattern:
A = .-
B = -...
SOS = ... --- ...
Importance
Morse code was used in ships, the military, and emergencies. It is still used today in some radio communications.
